# Sales-Dashboard
This project is an interactive Sales Dashboard built using Streamlit, Pandas, and Plotly.   It provides a clear overview of sales performance, payment status, and key business insights based on the dataset `Sample Data 2.xls`.

 Features

 Interactive filters (Payment Status, Source, Date Range)  
 KPIs: Total Sales, Paid Orders, Refund Amount, Average Order Value, Conversion Rate  
 Charts:
- Top 10 Products by Sales  
- Daily Sales Trend  
- Sales by Source  

 Auto-generated insights:
- Top Product  
- Best Performing Source  
- Conversion rate warnings  
- Refund alerts  

 View and Download Filtered Data as CSV
